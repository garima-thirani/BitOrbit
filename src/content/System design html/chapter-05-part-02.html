The Cache Mindset

Caching stores expensive-to-compute results in fast memory. Keeping cache and database in sync.

Caching Patterns: Keeping cache and database in sync.
Cache-Aside Pattern

1. Check cache first 2. Cache hit -> return 3. Cache miss -> query DB 4. Store in cache 5. Return data. This is the most common pattern.
